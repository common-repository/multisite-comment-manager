View and track all the pending comments in a Multisite.

== Description ==

Multisite Comment Manager is an easy-to-use tool that allows you to overview all your pending comments from across your Multisite network. You can always be up to date, when comments have been posted on your sites by checking the Multisite Comment Manager plugin dashboard.

= Check out the premium add-on features! =

- Approve comments
- Reject comments
- Edit comments
- ...All of the above from the main Multisite Comment Manager dashboard

Please send an email for details about the premium addon: <tettinger.solutions@gmail.com>

== Installation ==

= From within dashboard (recommended) =

1. Navigate to `Dashboard` ▸ `Plugins` ▸ `Add New`;
2. Search for `Multi Site Comment Manager`;
3. Click `Install`, then `Activate`.
4. The Comment Management tab will be located in your Main site's dashboard.

= Manual installation =

1. Download the plugin as a .zip file;
2. Unzip downloaded archive and upload `multisite-comment-manager` folder under your `/wp-content/plugins/`.
3. Navigate to `Dashboard` ▸ `Plugins` and activate the plugin.
4. The Comment Management tab will be located in your Main site's dashboard.
